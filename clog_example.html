CLog Demo

Welcome to CLOG! CLog (short for Console Log), is a JS logging function that works with the FireBug API to see logs in a vivid, clear way. Open FireBug and make sure your console is open before running this demo. Use the error log levels to change how 'deep' you want the messaging to go.

  1. No logging
  2. Errors only
  3. Errors and Warnings
  4. All messages are logged (counts are diabled)
  5. Full functionality
